# LeetCode Solutions

A collection of **LeetCode problem solutions** implemented in **Python**, organized by difficulty.  
This repository is intended for **learning, practice, and reference purposes**.

## Repository Structure

leetcode-solutions/
│
├── python/
│ ├── easy/ # Easy level problems
│ ├── medium/ # Medium level problems
│ └── hard/ # Hard level problems
│
├── README.md # This file
└── LICENSE

## Usage

1. Navigate to the `python/` folder.
2. Choose the difficulty level (`easy/`, `medium/`, `hard/`).
3. Open the solution file for the specific problem.
4. Run the file using Python 3.x:

```bash
python problem_name.py
